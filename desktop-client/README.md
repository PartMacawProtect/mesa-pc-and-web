# Mesa — Настольный клиент для ПК (Electron)

Данная папка содержит автономный Electron-клиент для мессенджера Mesa. При скачивании с вашего сайта архив уже полностью настроен на работу с вашим сервером через готовый файл `.env`.

## 🚀 Быстрый запуск

1. **Распакуйте** скачанный архив `Mesa-Electron-Client.zip` в любую удобную папку на вашем компьютере.
2. Убедитесь, что у вас установлен **Node.js** (скачать можно с [официального сайта nodejs.org](https://nodejs.org/)).
3. Откройте терминал (командную строку) в этой папке и выполните:

   ```bash
   # 1. Установка всех необходимых компонентов
   npm install

   # 2. Запуск настольного приложения
   npm start
   ```

## ⚙️ Настройки сервера (Переменные окружения)

В папке клиента уже создан и заполнен файл `.env`. При необходимости изменить адрес сервера (например, если вы перенесете сервер), просто откройте файл `.env` в текстовом редакторе и измените переменную `APP_URL`:

```env
APP_URL=https://mesa-pc-and-web-production.up.railway.app
```

## 📦 Сборка готового приложения (.EXE / .APP / installer)

Чтобы скомпилировать приложение в один исполняемый файл для вашей операционной системы (например, независимый `.exe` или `.dmg`):

```bash
# Сборка пакета под вашу ОС
npm run dist
```

Готовые файлы установки появятся в созданной папке `dist-desktop/`.
